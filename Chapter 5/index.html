Some More Array Methods

    1. Map
  • Creates a new array with the results of some operation. The value its callback returns are used to form new array
  • forEach is used when need to do normally a calculation or to print something and MAP is used when we need to create a new array from values

    • 2. Filter
  • Creates a new array of the elements that give true for a condition/filter.
    Eg.all even elements

    • 3. Reduce
  • Performs some operation & reduces the array to as single value. It returns that single value
    Eg. Sum of many numbers,average value
