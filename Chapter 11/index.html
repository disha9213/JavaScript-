Classes and Objects

Prototypes in JS

  • A JavaScript object is an entity having state and behaviour(properties and method)
  • JS objects have a special property called prototype.
  • We can set prototype using __proto__

Classes in JS

  • Class is a program-code template for creating objects
  • Those objects will have some state(variables) & some behaviour (functions) inside it
  • Syntax
    • class myClass{
               constructor(){...}
               myMethod(){....}
        }
let myObj = new MyClass();
  • Constructor() method is:initializes object and automatically invoked by new

Inheritance

  • Inheritance is passing down properties & methods from parent class to child class
    • class Parent{
    }
    class Child extends Parent{}
  • *If child and parent have same method ,childs method will be used

super keyword

  • The super keyword is used t call the constructor of its parent class to access the parent's properties and methods
    • super(args) //calls Parents Constructor
    super.parentMethod(args)

Error Handling

  • error in a middle code will block the execution for the rest of the code therfore try and catch statement is used
    • try-catch 
    try{
    ....normal code
}catch(err){  //err is error object
    ....handling error
